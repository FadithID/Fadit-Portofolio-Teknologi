Tentang Saya

Hai! Nama saya Fadit. Saya adalah seseorang yang sangat menyukai dunia coding dan teknologi, serta punya hobi bermain game di waktu luang. Bagi saya, menulis kode adalah cara untuk menciptakan hal-hal baru, menyelesaikan masalah, dan menuangkan ide-ide kreatif menjadi sesuatu yang nyata. Ketertarikan saya pada coding dimulai sejak saya penasaran bagaimana sebuah website bisa bekerja. Dari situ, saya mulai belajar HTML, CSS, dan JavaScript, lalu berkembang ke berbagai teknologi seperti React, Node.js, dan bahkan backend development. Saya suka membangun website yang tidak hanya terlihat menarik, tapi juga fungsional dan user-friendly. Saya percaya bahwa belajar tidak pernah berhenti, apalagi di dunia teknologi yang terus berkembang. Maka dari itu, saya senang mengeksplorasi hal-hal baru, mengikuti perkembangan tools terbaru, dan sesekali ikut tantangan coding atau hackathon. Melalui website ini, saya ingin membagikan hasil karya saya, berbagi cerita seputar coding dan game, serta terhubung dengan teman-teman baru yang punya minat serupa.
